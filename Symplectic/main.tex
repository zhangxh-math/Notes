    \section{Symplectic Manifold}
    \begin{definition}
      A smooth manifold \(M\) is called symplectic if \(M\) is equiped with a nondegenerated closed 2-form \(\omega\) .
    \end{definition}
    Let \(f\) be a smooth function on \(M\).
    There exist a unique vector fields \(X_{f}\) such that 
    \[
        \omega(X_{f},-)=\dd f(-).
    \] 
    Then we can define a Poisson bracket on \(M\) by 
    \[
        \{f,g\} := \omega(X_{f},X_{g}).
    \]
